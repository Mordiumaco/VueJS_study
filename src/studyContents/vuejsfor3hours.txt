PWA 기반 Single Page App 을 제작하기 위한 화면 앞단 라이브러리 Vue.js
Vue 로 화면을 구성하기 위한 기본적인 개념을 학습하고 샘플을 활용하여 코드 제작
Vue의 주요 구성요소 (Component, Router,Resource, Templates)를 학습 및 실습


node.js 를 설치하는 이유
1. npm(node package manager)를 이용한 view project 를 자동으로 구성하는 부분이 있어서 필요함
2. 노드 서버를 개인적으로 올려서 예를 들어 로컬 서버를 올려서 구동시키기 위해 (간단하게)


LTS (long term sofeversion) : 장기지원 버젼

node를 설치할때 node package manager 도 같이 설치가 된다.
